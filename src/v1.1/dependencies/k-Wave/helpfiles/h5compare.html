h5compare

Compare the contents of two HDF5 files.

Syntax

h5compare(filename1, filename2)

Description

h5compare compares the datasets and attributes within two HDF5 files. Fields that are present in one file but not the other are reported as missing. For fields that exist in both files but have different values, the L_inf error is reported. All outputs are printed to the command line.

Inputs

filename1 filename of the first HDF5 file
filename2 filename of the second HDF5 file

See Also

h5write, writeMatrix
